# AI Career Coach Design Specification

## Overview
This specification outlines the design for the AI Career Coach feature set within the Job Matching application. The goal is to elevate the platform from a simple job board to an intelligent advisor that actively helps candidates improve their profiles and bridge skill gaps to secure their desired roles.

## Core Features

### 1. AI Resume Enhancer (Option C)
**Purpose:** Provide both immediate, actionable feedback during the CV upload process and a comprehensive, persistent report for long-term improvement.

**Workflow:**
- **Quick Scan (Upload Phase):**
  - When a candidate uploads their CV, the backend parses the data via Gemini.
  - Alongside the parsed data, Gemini returns a "Quick Scan" summary (e.g., overall score, 1-2 major formatting or content issues).
  - The UI displays this Quick Scan as a dismissible alert or modal before the user officially saves their profile.
- **Detailed Report (Profile Tab):**
  - A new tab named "Resume Analysis" (Phân tích CV) is added to the Candidate Profile page.
  - This tab displays the full, detailed feedback generated by the AI.
  - Feedback categories include: Formatting, Tone/Action Verbs, Quantifiable Achievements, and Specific Suggestions to rewrite specific bullet points.

### 2. Skill Gap Analysis (Option A)
**Purpose:** Help candidates understand exactly what they are missing to qualify for a *specific* job posting.

**Workflow:**
- **Job-Specific Analysis:**
  - On the Job Detail page (`/candidate/feed` or `/jobs/[id]`), a new button labeled "Analyze Skill Gap" (Phân tích kỹ năng cho công việc này) is added.
  - When clicked, the backend compares the candidate's parsed skills against the job's `required_skills` and `brief_description`.
  - The AI generates a report that highlights:
    1. **Matching Skills:** What the candidate already has.
    2. **Missing/Weak Skills:** What the job requires but the candidate lacks or is weak in.
    3. **Learning Path:** A short, actionable learning path recommending topics or specific types of courses to bridge the gap.
- **UI Presentation:**
  - The analysis is presented in a slide-out panel (Sheet) or a dedicated Dialog on the Job Detail page to keep the context intact.

## Architecture & Integration
- **Backend:** 
  - Add `POST /candidate-profiles/me/analyze-resume` endpoint for detailed resume analysis on demand or save it during parsing.
  - Add `POST /jobs/{id}/analyze-gap` endpoint that triggers a Gemini prompt comparing the user's profile and the job object.
- **Frontend:**
  - Add UI components for the Quick Scan modal.
  - Add "Resume Analysis" tab in `/candidate/profile`.
  - Add "Analyze Skill Gap" button and results display in the Job Detail view.

## Verification
- Ensure Gemini responses are structured (JSON) to properly map to frontend UI components (scores, lists of suggestions).
- Ensure stateless or minimal-state execution for ad-hoc analysis to avoid bloating the database unless necessary.
